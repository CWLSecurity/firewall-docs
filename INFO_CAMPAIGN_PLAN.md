# Info Campaign Plan (MVP)

## Phase 1 — Soft Launch
- 5–10 power users
- Collect issues/UX feedback
- Fix critical bugs

## Phase 2 — Public Launch
Channels:
- Twitter/X thread
- Telegram crypto communities
- Reddit (ethdev/defi)
- Short demo video + landing page

Messaging pillars:
- "Protected version of your wallet"
- "On-chain enforcement, not warnings"
- "No custody, no backend"

Metrics:
- Connects
- Wallet creations
- Imports
- Sends
- Execute/cancel actions
